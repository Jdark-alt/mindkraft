🔑 VAPID Key Generator

One-time setup for Mindkraft push notifications. Run this once, save the keys as GitHub Secrets, then delete this file.

⚠️ Save both keys somewhere safe right now. You cannot retrieve them again after closing this page. If you lose them, just regenerate and re-save your reminder in the app.
Step 1 — Generate keys
Click the button above. Copy both keys.
Step 2 — Add GitHub Secrets
Go to your GitHub repo → Settings → Secrets and variables → Actions → New repository secret.
Add these 4 secrets:
  • VAPID_PUBLIC_KEY — the public key above
  • VAPID_PRIVATE_KEY — the private key above
  • VAPID_CONTACT_EMAIL — your email address (e.g. you@gmail.com)
  • FIREBASE_SERVICE_ACCOUNT — your Firebase service account JSON (see Step 3)
Step 3 — Firebase Service Account
Go to Firebase Console → your project → Project Settings → Service accounts → Generate new private key.
Download the JSON file. Open it, select all text, and paste the entire content as the FIREBASE_SERVICE_ACCOUNT secret.
Step 4 — Paste public key into app.js
In app.js, find this line near the bottom:

var VAPID_PUBLIC_KEY = 'PASTE_YOUR_VAPID_PUBLIC_KEY_HERE';

Replace the placeholder with your actual public key (in quotes).
Step 5 — Push all files and test
Commit and push: index.html, app.js, sw.js, scripts/package.json, scripts/send-reminders.js, and .github/workflows/send-reminders.yml.

Then go to Actions tab → Send Daily Reminders → Run workflow to do a manual test run. Check the logs to confirm it ran without errors.
Step 6 — Delete this file
Remove vapid-keygen.html from your repo. It's served publicly on GitHub Pages and there's no reason to keep it online.
